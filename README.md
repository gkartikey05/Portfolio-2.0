<h2 id="setup--installation" align="left">âš¡ Setup & Installation</h2>

### Prerequisites

- Node.js & npm

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/gkartikey05/Portfolio-2.0.git
   cd Portfolio-2.0
   ```
2. **Install dependencies:**
   - Frontend:
     ```bash
     cd CLIENT
     npm install
     ```
   - Backend:
     ```bash
     cd ../SERVER
     npm install
     ```
3. **Run the project:**
   - Start backend server:
     ```bash
     npm start
     ```
   - Start frontend (in CLIENT):
     ```bash
     npm run dev
     ```
